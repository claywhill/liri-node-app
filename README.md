## Usage
1. Clone the repo
1. npm install axios, Node-Spotify-API, Moment, DotEnv
1. Get our own Spotify API key
1. Copy your API key into .env
   * SPOTIFY_ID=your-spotify-id
   * SPOTIFY_SECRET=your-spotify-secret
